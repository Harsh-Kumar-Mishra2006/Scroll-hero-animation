## ‚ú® Overview

Scroll Hero Animation is a stunning, interactive web application that takes users on a visual journey through the world of Artificial Intelligence. From its historical origins to future predictions, from mind-bending statistics to real-world applications - every page is designed to educate, inspire, and delight.

The project features **cinematic scroll animations**, **floating 3D elements**, **dynamic statistics that change as you scroll**, and a **beautiful tech-inspired aesthetic** with glass morphism, glowing effects, and particle backgrounds.

### üè† Home Page
